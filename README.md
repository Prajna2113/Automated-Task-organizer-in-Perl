# Automated-Task-organizer-in-Perl
Object-Oriented Task Organizer built in Perl with file handling and priority sorting.


## Overview
This is a Command Line Task Organizer built using Perl.
The project demonstrates Object-Oriented Programming concepts,
file handling, sorting algorithms, and data persistence.

## Features
- Add tasks with title, description, due date, and priority
- View all tasks
- Delete tasks
- Sort tasks by priority (High → Medium → Low)
- Save tasks to file
- Persistent storage using text file

## Technologies Used
- Perl
- OOP Concepts (Packages, Constructors, Methods)
- File Handling
- Sorting using custom comparator

## Concepts Implemented
- Object-Oriented Programming in Perl
- Hash-based object structure
- File parsing using regex
- Multi-level sorting
- CLI interaction
